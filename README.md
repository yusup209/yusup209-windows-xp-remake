## Demo

Open `index.html` in a web browser to experience the Windows XP desktop recreation.

## Usage

1. **Desktop Icons**: 
   - Single-click to select
   - Double-click to open (shows informational alert)

2. **Start Menu**: 
   - Click the Start button to open
   - Click menu items to interact
   - Click outside to close

3. **Taskbar**: 
   - View real-time clock in system tray
   - Click system icons (decorative)

## Technical Details

- **Pure vanilla JavaScript** - No frameworks or libraries required
- **CSS Grid layout** - For responsive desktop icon positioning
- **SVG icons** - Scalable vector graphics for crisp rendering at any size
- **CSS gradients** - Authentic Luna theme appearance
- **Event handling** - Interactive functionality throughout
- **Real-time clock** - Updates every second
